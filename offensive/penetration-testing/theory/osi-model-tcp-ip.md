# OSI Model - TCP/IP

## OSI Model

```
1 : Physical (Bits)
2 : Data Link (Frames)
3 : Network (Packets)
4 : Transport (Segments)
5 : Session (Data)
6 : Presentation (Data)
7 : Application (Data)
```

## TCP/IP Model

```
1 : Network Interface
2 : Internet Layer
3 : Transport Layer
4 : Application Layer
```
