# VLAN

* 802.1Q Trunking
* VLAN IDs - 12 bits long, 4094 VLANs
* Normal Range 1002-1005
* Extended Range 1006-4094
* 0 and 4095 are reserved VLAN numbers
* VLAN.dat - Configuration for VLAN IDs 1-1005
